#Calculadora de IMC

Esse projeto tem o objetivo de calcular seu índice de IMC, para utiliza-lo é só abrir a página, digitar seu peso e altura e clicar em calcular

**Link da página**

https://izaqueisrael.github.io/imcCalculator/

**Tecnologias Usadas**

HTML para o corpo, css para a estilização e javascript para realizar os calculos e a adição do resultado na página
